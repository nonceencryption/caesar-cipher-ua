# 🏛️ Caesar Cipher Tool (UA Edition)

Цей скрипт створений для демонстрації роботи класичного шифру Цезаря, адаптованого під український алфавіт (33 літери, включаючи 'ґ' та 'є'). 

## 🛠️ Можливості:
1. **Зашифрувати:** Зміщує текст на задану кількість позицій (вправо).
2. **Розшифрувати (Brute Force):** Автоматично перебирає всі 32 можливі варіанти зсуву. Залишається лише знайти очима читабельний текст.

## 🚀 Як запустити:
Переконайся, що в тебе встановлений Python. Відкрий термінал у папці зі скриптом і напиши:
\`\`\`bash
python caesar.py
\`\`\`

## 🧠 Приклад роботи Brute Force:
Якщо закинути зашифрований текст `Тдолпр Нухпоя р 12:00`, скрипт видасть:
* Зсув -1: Сгнкпо Мтфоню п ...
* Зсув -2: Рвмйон Лсунмь о ...
* Зсув -3: **Палимо Кремль о 12:00** ✅

Користуйтесь! 🇺🇦
